# Examples

## Example 1: Classify a collection

User asks:
- 这 50 套 HTML 模板该怎么分组？

Expected response shape:
1. classify by use case, not by color
2. identify dominant groups such as corporate, portfolio, blog, product, and industry
3. select representative samples instead of listing everything equally

## Example 2: Pick reusable samples

User asks:
- 哪几套最值得做后续技能素材？

Expected response shape:
1. choose a small high-value sample pool
2. explain why each one is structurally representative
3. point out what kind of site each sample best supports

## Example 3: Extract sections

User asks:
- 这套老模板里哪些部分值得留下来？

Expected response shape:
1. identify reusable sections such as hero, features, gallery, and footer
2. separate structure value from outdated code
3. recommend reuse of pattern, not blind copy of legacy implementation
