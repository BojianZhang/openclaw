# Section Extraction

Use this file when pulling reusable layout ideas from static HTML templates.

## Common sections to extract

- header and navigation
- hero section
- feature grid
- services block
- portfolio or gallery grid
- testimonial section
- pricing section
- blog preview section
- contact section
- footer

## Extraction rule

Extract by structure and purpose, not by copied markup alone.
Ask:
- what job does this section do?
- what makes its hierarchy effective?
- what is reusable even if the old CSS or JS is outdated?

## Practical reminder

The goal is to reuse layout patterns and information architecture, not inherit unnecessary legacy code, plugin debt, or outdated browser hacks.
