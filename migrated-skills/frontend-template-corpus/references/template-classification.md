# Template Classification

Use this file to classify static frontend templates by primary use case rather than by superficial color or branding.

## Primary classes

- corporate website
- agency website
- portfolio
- photography or visual showcase
- blog or content-heavy site
- product showcase
- e-commerce or shop
- industry-specific website
- landing page or one-page promo
- coming soon page

## Classification rule

Choose the class that best matches the template's main structural intent.
Many templates are hybrid. In that case, classify by the strongest user-facing purpose, then note secondary traits.

## Practical signals

Look for:
- navigation structure
- page count
- repeated section types
- portfolio or gallery density
- blog or article page presence
- product or pricing emphasis
- contact and CTA prominence

## Warning

Do not classify only by screenshot style. Structure and page purpose matter more than color palette.
