# 章节级知识卡：Java 并发编程实战 核心章法

## 主题 1：线程安全本质
### 核心点
- 并发问题 = 共享可变状态 + 时序不确定
- 三大分析维度：原子性、可见性、有序性
- 少共享比多加锁更优先

## 主题 2：synchronized / volatile / 原子类
### 核心点
- synchronized 解决互斥 + 可见性
- volatile 主要解决可见性，不保证复合操作原子性
- 原子类适合小粒度无锁更新，但不替代完整并发设计

## 主题 3：线程池与任务模型
### 核心点
- 线程池是资源边界，不是性能开关
- 先区分 CPU 密集 / I/O 密集任务
- 队列容量、拒绝策略、超时策略都要设计

## 主题 4：发布与逸出
### 核心点
- 对象构造未完成就暴露出去很危险
- this 逸出、可变对象共享、懒加载都要谨慎

## 主题 5：取消与中断
### 核心点
- 取消不是强杀线程，而是协作式停止
- 中断要么传递，要么恢复中断标记，不能随便吞掉
