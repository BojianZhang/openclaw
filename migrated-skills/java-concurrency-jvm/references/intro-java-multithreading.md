# 深入浅出 Java 多线程 - 深摘要

## 这本书解决什么问题
它解决的是：**如何用更容易理解的方式建立 Java 多线程直觉。**

## 核心主线
1. 线程与进程区别
2. 线程状态与生命周期
3. synchronized、wait/notify、volatile 基础语义
4. 常见并发协作模型

## 最关键的知识骨架
### 1. 多线程最难的不是 API，而是时序
很多 bug 都来自：
- 你以为 A 先发生，实际上 B 先发生
- 你以为变量已经更新，另一个线程却没看见

### 2. 先建立直觉，再上高级工具
如果基础线程模型没搞清楚，线程池、并发容器、CompletableFuture 都会学得漂。
