# 后台系统模拟答法模板

当用户要围绕后台系统项目做模拟面试时，优先使用这份模板。

## 1. 统一开场模板
我做过的后台系统项目主要分三类：一种偏企业经营流程，比如进销存；一种偏客户管理和运营，比如 CRM；还有一种偏前后端分离平台能力，比如云E办。它们都属于后台系统，但各自的问题域不同，所以我会按面试问题去选最合适的样本，而不是把所有后台项目讲成一套套路。

## 2. 如果问题偏业务闭环
优先切到：`进销存（JXC）`

回答重点：
- 采购、销售、库存、财务闭环
- 单据流转
- 库存变化
- 报表和后台管理

## 3. 如果问题偏客户管理/运营
优先切到：`CRM`

回答重点：
- 客户管理
- 营销过程
- 权限管理
- 报表统计
- 客户生命周期

## 4. 如果问题偏平台协同/权限安全
优先切到：`云E办（YEB）`

回答重点：
- 前后端分离
- Spring Security / OAuth2 / JWT / Redis token
- RabbitMQ / Redis
- SSO
- 平台工程化与协同开发

## 5. 收束句
这三类项目我不会混着讲，而是按问题域来切样本：问业务流程我讲进销存，问客户运营我讲 CRM，问前后端分离和权限体系我讲云E办。
