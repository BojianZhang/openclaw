# CRM 项目答题模板

当用户要讲客户管理/营销管理/权限管理/报表统计类项目时，优先使用这份模板。

## 1. 你做过什么项目
我做过一个 CRM 客户管理系统，核心围绕客户管理、营销管理、权限管理和统计报表展开。这个项目不是电商交易平台，也不是进销存系统，它更偏客户生命周期管理和运营视角。技术上是 Spring Boot + MyBatis + FreeMarker 这类典型后台系统结构。

## 2. 这类项目的价值怎么讲
它的价值在于把客户开发、客户信息维护、营销过程管理、权限控制和统计分析串起来，体现的是客户管理和业务运营能力，而不是交易链或高并发能力。

## 3. 为什么权限和报表值得单独讲
因为 CRM 不只是录客户信息，真正有价值的是不同角色能看到什么、能做什么，以及管理层怎么通过统计报表看客户和营销效果。

## 4. 项目连接句
- “这个项目更偏客户管理和运营后台，而不是电商交易平台。”
- “我会重点讲客户、营销、权限、报表这几条主线。”
- “它最值的地方是客户生命周期和运营视角，而不是中间件复杂度。”
