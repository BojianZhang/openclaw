# 电商项目模拟答法模板

当用户要围绕电商项目做模拟面试时，优先使用这份模板。

## 1. 统一开场模板
我接触过的电商项目大致可以分成四类：一种是单体购物链项目，比如 Ebuy；一种是模块化平台项目，比如乐SHOP；一种是传统分布式商城，比如 易购商城；还有一种是高并发交易链专项，比如双11秒杀方案。它们都属于电商语境，但问题域完全不同，所以我会根据面试题来切最合适的样本。

## 2. 如果问题偏单体购物流程
优先切到：`Ebuy`

## 3. 如果问题偏平台边界和支撑能力
优先切到：`乐SHOP`

## 4. 如果问题偏传统 RPC 商城协作
优先切到：`易购商城`

## 5. 如果问题偏高并发交易链
优先切到：`双11秒杀方案`

## 6. 收束句
问普通购物链我讲 Ebuy，问平台能力我讲乐SHOP，问 Dubbo/Solr/传统分布式我讲 易购商城，问高并发秒杀我讲双11秒杀方案。
