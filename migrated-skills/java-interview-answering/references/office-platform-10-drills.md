# 前后端分离办公平台 10 题实战演练

当用户要围绕云E办这类前后端分离办公平台做实战问答时，优先使用这份演练集。

## 使用顺序
1. 先判断问题是在讲业务模块，还是平台能力
2. 先用一句话定义项目类型
3. 再按“前端协同 / 后端接口 / 权限安全 / 缓存消息 / 工程化”展开

## 10 个高频问题
1. 你做过什么前后端分离项目？
2. 这个项目和传统后台系统有什么区别？
3. 为什么要用 Vue + Spring Boot？
4. 这个项目里的权限体系怎么讲？
5. Spring Security 在项目里起什么作用？
6. OAuth2 / JWT / Redis token 这条线该怎么解释？
7. RabbitMQ 在办公平台里适合放在哪些场景？
8. Redis 在这类项目里更适合承接什么？
9. 你怎么体现前后端协同开发能力？
10. 如果面试官问这个项目最大的价值，你怎么总结？

## 默认答题骨架
- 先定义：这是前后端分离办公/人事平台，不是传统单体后台
- 再讲平台能力：权限、安全、缓存、消息、协同
- 最后讲业务：组织、角色、办公流程等模块
