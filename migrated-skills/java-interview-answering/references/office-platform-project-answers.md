# 前后端分离办公平台项目答题模板

当用户要讲云E办这类 Vue + Spring Boot 办公/人事管理平台时，优先使用这份模板。

## 1. 你做过什么项目
我做过一个前后端分离的办公管理平台，后端基于 Spring Boot，前端基于 Vue2 + Element UI，项目里还配套用了 Redis、RabbitMQ、Spring Security、OAuth2、JWT、SSO 等能力。这个项目更像企业内部办公/人事管理平台，重点在权限体系、组织管理、消息通知和前后端协作。

## 2. 这类项目的价值怎么讲
它的价值不只是后台 CRUD，而在于前后端分离架构、权限认证体系和协同能力比较完整。前端负责路由、状态管理和界面交互，后端负责业务接口、认证授权、缓存、消息和工程化支撑。

## 3. 为什么安全体系值得讲
因为这类平台通常涉及登录、角色权限、单点登录、token 管理等问题。Spring Security、OAuth2、JWT、Redis token 存储这类能力，不是点缀，而是平台可用性的核心部分。

## 4. 为什么前后端协作值得讲
因为它能体现你不仅会写后端接口，也理解 Vue、Axios、Vue Router、Vuex、Element UI 这套前端管理平台协作方式。

## 5. 项目连接句
- “这个项目更像前后端分离办公平台，而不是传统单体后台。”
- “我会重点讲权限认证、组织管理、消息协同和前后端配合。”
- “它比普通后台项目更值的地方，在于安全体系和前后端协同完整。”
