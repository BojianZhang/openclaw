# 图解 TCP/IP / 计算机网络：自顶向下方法 / 网络是怎样连接的 - 深摘要

## 这组书解决什么问题
它们解决的是：**如何先建立网络分层与传输直觉，再理解更复杂的 TCP/HTTP/网络故障。**

## 核心主线
1. 网络分层模型
2. IP、TCP、UDP 的角色分工
3. 路由、寻址、传输、应用层协议
4. 从“网络是黑盒”过渡到“网络是可分析系统”

## 最关键的知识骨架
### 1. 网络问题必须先建立分层视角
很多问题如果不区分层级，就会混成一团。

### 2. 直觉层很重要
在学习 TCP/IP 详解之前，先能直观理解连接、路由、报文、握手、重传，会大幅降低学习成本。
