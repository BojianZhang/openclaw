# 实战 Java 虚拟机 - 深摘要

## 这本书解决什么问题
它解决的是：**JVM 问题在真实线上环境中应该怎么观察、判断和调优。**

## 核心主线
1. 常见线上 JVM 症状
2. GC 和内存问题分析
3. 参数调优与日志观察
4. 故障定位方法

## 最关键的知识骨架
### 1. 原理要落到证据
JVM 排障不能只停留在概念上，要看：
- GC 日志
- 堆使用趋势
- 线程栈
- 实际压测/线上现象

### 2. 参数调整必须和现象对应
不是“别人这么配我也这么配”，而是要知道：
- 为什么调
- 调了会影响什么
- 副作用是什么

## 工程落地价值
### 适合回答的问题
- 线上 Full GC 怎么查？
- 堆参数怎么调更合理？
- 为什么服务偶发抖动？
