# 章节级知识卡：Java 核心技术（卷 1 / 卷 2）核心章法

## 主题 1：语言与标准库基础
### 核心点
- 集合、异常、泛型、I/O、反射是 Java 工程开发的底座
- 框架问题最后经常要回到底层语言和 API 理解

## 主题 2：对象与类型系统
### 核心点
- 面向对象、继承、多态、接口、泛型共同构成 Java 抽象能力基础
- 基础薄弱会直接影响后续框架设计与排障能力

## 主题 3：高级主题扩展
### 核心点
- 网络、国际化、并发补充、反射等高级主题扩展 Java 知识边界
- 这套书适合作为系统查漏补缺框架

## 主题 4：工程意义
### 核心点
- 它不是单点最深，但非常适合建立完整 Java 知识地图
