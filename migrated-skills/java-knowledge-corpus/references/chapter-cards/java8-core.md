# 章节级知识卡：Java 8 实战 核心章法

## 主题 1：Lambda 与函数式接口
### 核心点
- 用行为参数化替代部分样板代码
- 函数式接口是 Java 8 风格的基础

## 主题 2：Stream
### 核心点
- map / filter / reduce 是核心骨架
- Stream 更适合数据处理流水线，不是所有逻辑都适合链式写法

## 主题 3：Optional
### 核心点
- Optional 用于显式表达“可能没有值”
- 不适合滥用到一切边界和字段上

## 主题 4：新的时间 API
### 核心点
- 用更清晰不可变的时间模型替代旧 Date/Calendar 痛点
