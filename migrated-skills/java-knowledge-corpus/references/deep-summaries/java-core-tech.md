# Java 核心技术（卷 1 / 卷 2）- 深摘要

## 这套书解决什么问题
它解决的是：**如何系统掌握 Java 标准知识体系，而不是只会片段式写代码。**

## 核心主线
1. Java 基础语言机制
2. 标准库与常见 API
3. 高级主题补充
4. 用完整知识面支撑工程开发

## 最关键的知识骨架
### 1. 建立“Java 知识地图”
这套书的价值不在某个单点特别深，而在于它能让你知道：
- Java 世界里有哪些基础块
- 它们之间是怎么关联的

### 2. 基础不牢会拖累后续所有高级主题
并发、JVM、框架、微服务最后都会落回：
- 集合
- 类型系统
- 异常
- I/O
- 反射
- 网络基础 API

## 工程落地价值
### 适合回答的问题
- Java 基础知识体系怎么补全？
- 标准库该怎么系统掌握？
- 为什么高级框架问题最后总要回到底层语言基础？

## 常见误区
- 误以为基础书过时就不用看
- 误以为会用框架就等于懂 Java

## 与其他书的连接
- 它是很多高级书的底座
- 适合与 Effective Java、并发、JVM、数据库书配套学习
