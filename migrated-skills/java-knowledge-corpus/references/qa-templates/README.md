# QA Templates

用途：用于 **宽口径、高频、主题级** 问题的快速回答。

当前覆盖：
- Java 设计与代码质量
- Spring Boot 工程实践
- MySQL 与 Redis
- 网络与微服务

使用原则：
1. 只有在用户问题是高频、宽口径、主题级问答时才优先读本目录。
2. 如果问题已经进入排障、调优、实现或机制深水区，优先转 specialist skill，不在这里继续下钻。
3. 如果 `topic-cards/` 已经足够回答，就不要为了补细节默认加载本目录。
4. 本目录是总入口补充层，不是 specialist checklist、decision card 或 implementation template 的替代品。
