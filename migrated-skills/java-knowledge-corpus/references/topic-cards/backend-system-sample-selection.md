# 专题卡：后台系统样本选择

## 适用问题
- 进销存、CRM、云E办 这三类后台系统样本该怎么选？
- 如果面试官问的是后台系统项目，我该优先讲哪套？

## 选择原则
### 1. 问业务闭环
优先：`进销存（JXC）`

关键词：采购、销售、库存、财务、单据流转、报表

### 2. 问客户运营 / 营销 / 报表
优先：`CRM`

关键词：客户管理、营销管理、权限、客户开发、统计分析

### 3. 问前后端分离 / 权限认证 / 平台能力
优先：`云E办（YEB）`

关键词：Vue、Spring Boot、Security、OAuth2、JWT、Redis、RabbitMQ、SSO

## 常见误区
- 不要把 CRM 讲成进销存
- 不要把进销存讲成通用后台平台
- 不要把云E办只讲成普通 CRUD 后台

## 快速口诀
- 问流程闭环 -> 进销存
- 问客户运营 -> CRM
- 问平台协同和安全 -> 云E办
