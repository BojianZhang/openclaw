# 专题卡：Java 面试准备

## 适用问题
- Java 面试怎么准备？
- 八股怎么答更有结构？
- 算法和项目经验怎么一起准备？

## 核心知识来源
- 剑指 Offer
- 程序员面试宝典
- Effective Java
- Java 并发编程实战
- 深入理解 Java 虚拟机
- MySQL 高性能
- Redis 入门指南
- HTTP 权威指南

## 核心结论
1. 面试准备要分层：语言 -> 并发/JVM -> DB/缓存/网络 -> 项目经验
2. 重点不是只会背，而是能结构化表达
3. 项目经历和排障经验会让答案明显更有说服力

## 常见误区
- 只刷题不整理知识框架
- 只背八股不结合项目
- 回答没有结构，只会零散堆点

## 推荐回答模板
- 一分钟短答
- 三分钟展开
- 再补项目里的实际例子
