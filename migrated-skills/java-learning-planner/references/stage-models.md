# Stage Models

## 1. 零基础到 Java 入门
目标：建立语言基础与面向对象直觉。
优先：Java 核心技术、Java 编程思想、基础集合与异常。

## 2. Java 入门到后端入门
目标：把语言知识接到 Web/数据库/HTTP。
优先：Spring Boot 基础、MySQL、Redis、HTTP。

## 3. 初级后端到进阶后端
目标：补并发、JVM、缓存、一致性、工程结构。
优先：并发编程实战、深入理解 JVM、MySQL 高性能、重构。

## 4. 在职提升
目标：解决真实项目里的性能、稳定性、结构与治理问题。
优先：JVM、数据后端、微服务治理、设计与重构。

## 5. 面试冲刺
目标：把知识框架压缩成可表达答案。
优先：高频模块梳理 + 项目表达 + 八股结构化输出。
