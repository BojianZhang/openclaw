# 章节级知识卡：HTTP 权威指南 核心章法

## 主题 1：HTTP 语义
### 核心点
- 方法、状态码、Header 都有明确语义
- REST 不是“返回 JSON”这么简单
- 先理解协议语义，再设计接口

## 主题 2：连接与性能
### 核心点
- Keep-Alive、连接复用、代理转发都会影响系统表现
- HTTP 问题经常和 TCP/代理层交织

## 主题 3：缓存与 Cookie
### 核心点
- 缓存控制决定一致性和性能边界
- Cookie / Session 不是纯前端问题，而是链路状态管理问题

## 主题 4：排障视角
### 核心点
- 请求异常时先区分协议层、代理层、应用层
- 状态码要和语义对应，不能乱用
