# 章节级知识卡：UNIX 网络编程 核心章法

## 主题 1：Socket 模型
### 核心点
- 网络编程最终都落回 socket、连接、读写与关闭
- Java 的 HTTP/Netty/Tomcat 问题，很多都能映射到 socket 行为理解

## 主题 2：阻塞 / 非阻塞 / 多路复用
### 核心点
- I/O 模型决定线程利用率和连接处理方式
- 性能问题常和 I/O 模型选择有关

## 主题 3：连接生命周期
### 核心点
- connect / accept / read / write / close 的顺序与异常行为非常关键
- 很多超时与 reset 问题来自连接生命周期理解不清

## 主题 4：工程意义
### 核心点
- 它是理解 Java 网络服务底层行为的核心桥梁
