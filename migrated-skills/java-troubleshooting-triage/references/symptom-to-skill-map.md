# Symptom to Skill Map

## 启动失败 / 配置不生效 / Bean 报错
转 `java-spring-microservice`

## 高 CPU / Full GC / OOM / 延迟抖动
转 `java-jvm-performance`

## 线程池打满 / 死锁 / JMM / 并发阻塞
转 `java-concurrency-jvm`

## MySQL 慢 / Redis 失效 / 缓存一致性
转 `java-data-backend`

## timeout / reset / keep-alive / RPC 链路不稳
转 `java-network-microservices`

## 结构混乱 / 职责边界不清 / 改一处炸一片
转 `java-design-refactor`
