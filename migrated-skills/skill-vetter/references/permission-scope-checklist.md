# Permission Scope Checklist

## 要检查什么

### 文件范围
- 它需要读哪些文件？
- 它需要写哪些文件？
- 是否会修改工作区外文件？
- 文件访问范围是否和宣称功能一致？

### 命令范围
- 它会运行哪些命令？
- 是否会安装包、拉取远程代码、执行 shell 脚本？
- 是否需要 elevated / sudo？

### 网络范围
- 是否需要联网？
- 联到哪里？域名是否清楚？
- 是读取公开资料，还是上传数据？

### 数据范围
- 是否会读取用户隐私、令牌、记忆、配置、凭证？
- 是否会把本地内容发到远端？

## 判断原则
- 权限越宽，风险越高。
- “功能合理”不等于“权限合理”。
- 如果 scope 不能被一句话说清，默认不是最小权限。
