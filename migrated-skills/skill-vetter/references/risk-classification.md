# Risk Classification

## 🟢 LOW
适合：
- 纯文本整理
- 笔记、格式化、天气、只读型小工具

动作：
- 做基础审查后可安装

## 🟡 MEDIUM
适合：
- 文件操作
- 浏览器自动化
- 普通 API 访问
- 有脚本执行但边界清晰

动作：
- 做完整代码审查
- 核对权限范围

## 🔴 HIGH
适合：
- 凭证相关
- 系统配置
- 自动安装依赖
- 广泛文件访问
- 网络写操作

动作：
- 需要人工确认
- 默认谨慎，不清楚就不装

## ⛔ EXTREME
适合：
- root / sudo / elevated
- 安全配置修改
- 凭证、cookie、session 抓取
- 模糊用途的大范围系统访问

动作：
- 不安装

## 提升风险等级的条件
- 来源不明
- 作者不明
- 文件多且复杂
- 有隐藏脚本
- 用途与权限不匹配
- 存在混淆或远程下载执行
