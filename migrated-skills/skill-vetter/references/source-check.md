# Source Check

## 先查来源
- 来自 ClawdHub、GitHub，还是别的来源？
- 作者是否已知？
- 最近是否维护？
- 是否有下载量、star、review、历史记录？

## 来源层级
1. 官方 OpenClaw / 明确可信来源
2. 高 star / 高下载 / 已知作者
3. 普通公开仓库
4. 新号、匿名、来源不明
5. 私发压缩包、临时链接、不可追溯来源

## 快速检查建议
- 仓库主页
- 最近更新时间
- 文件数量与结构
- 是否有隐藏目录、脚本、hook
- 是否存在二进制、压缩包、混淆文件

## 原则
- 来源可信只能降低怀疑，不等于免审。
- 来源差时，即使代码表面干净，也提高风险等级。
